Sistema digital para gestão e organização de fluxos institucionais e acadêmicos. **Este projeto está sendo desenvolvido como parte das atividades práticas da disciplina de Programação Web I, do curso de Análise e Desenvolvimento de Sistemas (TADS) no IFMT.**

A construção do sistema tem como foco principal o estudo e a aplicação real do padrão arquitetural **MVC (Model-View-Controller)**, garantindo uma arquitetura escalável no back-end, integridade no fluxo de dados e uma comunicação fluida com as interfaces web.

## ✨ Funcionalidades (em desenvolvimento)
* **Arquitetura MVC:** Separação clara entre a lógica de negócios, o acesso a dados e a interface do usuário.
* **Gestão de Entidades:** Cadastro, leitura, atualização e exclusão (CRUD) de dados institucionais.
* **Interface Responsiva:** Views estruturadas com HTML5 e CSS3 para uma navegação limpa e intuitiva.

## 🛠️ Tecnologias Utilizadas
* **Back-end:** C# e ASP.NET Core MVC
* **Front-end:** HTML, CSS e Razor Views
* **Padrão de Projeto:** MVC (Model-View-Controller)

## Como executar o projeto

**1. Clone este repositório:**
*git clone [https://github.com/afloreto/academico-web.git](https://github.com/afloreto/academico-web.git)*

**2. Abra a pasta do projeto na sua IDE preferida (Visual Studio ou VS Code).**

**3. Restaure as dependências do projeto:**
*dotnet restore*

**4.Execute a aplicação:**
*dotnet run*
